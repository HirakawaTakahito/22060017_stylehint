# システムの目的
ユーザーの好みをもとに、失敗しにくいファッション選びをサポート！
# システムのユニーク性
一般的なファッションアプリが「流行」や「ブランド」を基準に提案するのに対し、本システムはサイズ感や着用感、柄・色といったユーザーの主観的な好みを基準にスタイルを提案する点にユニーク性がある。
