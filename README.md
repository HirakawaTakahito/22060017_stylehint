# チーム4 モリモリ筋肉ーず
平川嵩飛人、久我祐司、岩隈大悟
# システムの目的
手持ちのアイテムで着こなしで悩んでる人、服を買っても着回しができない・同じ服ばかりになってしまう人にユーザーの好みをもとに、失敗しにくいファッション選びをサポート！
# システムのユニーク性
一般的なファッションアプリが「流行」や「ブランド」を基準に提案するのに対し、本システムはスタイリスト・ファッション誌・人気インフルエンサーのスタイルデータをもとに学習したRAGモデルを利用し、ユーザー個人のワードローブに合わせた“プロ監修風”の提案が可能になってます。
